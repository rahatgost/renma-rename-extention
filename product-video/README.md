# Renma product video

A professional 48-second product video for Renma, built with Remotion.

## Scenes

1. Hero — brand promise and source-aware filename preview.
2. Problem — messy download names creators deal with.
3. Advanced controls — templates, domain rules, duplicate handling, private stats.
4. Premium UI — popup and stats dashboard.
5. Final CTA — organized Downloads folder.

## Render

```sh
npm install
npm run render
```

The output will be written to `out/renma-product-video.mp4`.
